# KIYIM.UZ — интернет-магазин одежды

Демо-проект для портфолио: каталог, корзина, оформление заказа и оплата P2P-переводом. Сделано на React + Vite.

## Запуск

```bash
npm install
npm run dev
```

Откроется адрес вида `http://localhost:5173` — там магазин.

## Сборка для продакшена

```bash
npm run build
npm run preview
```

Готовые файлы для деплоя (например, на Vercel / Netlify / GitHub Pages) появятся в папке `dist`.

## Уведомления о заказах в Telegram

Когда покупатель подтверждает оплату, магазин отправляет заказ (товары, сумму, имя, телефон, адрес) в твой Telegram-чат. Без настройки магазин продолжит работать, просто уведомление отправляться не будет.

### 1. Создай бота

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`, придумай имя и username
3. BotFather пришлёт **токен** вида `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx` — это `VITE_TG_BOT_TOKEN`

### 2. Узнай свой chat_id

1. Найди в Telegram своего свежесозданного бота и нажми **Start** (или напиши ему любое сообщение)
2. Открой в браузере:
   `https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates`
3. В ответе (JSON) найди `"chat":{"id": ...}` — это число и есть `VITE_TG_CHAT_ID`

   Если ответ пустой — убедись, что ты действительно написал боту первым (боты не могут писать первыми).

### 3. Заполни .env

Скопируй `.env.example` в `.env` и вставь значения:

```
VITE_TG_BOT_TOKEN=123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
VITE_TG_CHAT_ID=987654321
```

Перезапусти `npm run dev` — теперь заказы будут прилетать тебе в Telegram.

> **Важно:** это фронтенд-проект без сервера, поэтому токен бота попадает в код, который выполняется в браузере покупателя (виден в devtools / собранном JS). Для учебного портфолио это нормально, но для реального магазина с деньгами такой токен лучше прятать за маленьким серверным прокси (например, Vercel Serverless Function или Cloudflare Worker), чтобы никто не мог использовать его от твоего имени.

## Стек

- React 18
- Vite
- lucide-react (иконки)

Все стили встроены прямо в компонент `src/App.jsx` (без Tailwind и внешних CSS-фреймворков), шрифты подключаются из Google Fonts.
